PCB-Wound Electromagnet Design Automation

This project focuses on the use of MATLAB to automatically generate code to import into KiCAD for rectangular, planar helix electromagnets. The parameters for these electromagnets can be varied to create electromagnets of arbitrary size, current capacities, and layer count. Additionally, the resultant magnetic field of these electromagnets can be calculated and saved to an external file.
