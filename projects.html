Regenerative Braking Trailer

Learn more

Inspired by the need for landscapers to navigate stop-and-go traffic on a daily basis, this project aimed to make use of all the energy lost to braking during a long day of driving through stop-and-go traffic. With the Regenerative Trailer Braking System, we aimed to make use of braking energy and reduce the wear on a vehicle's brakes by activating regenerative braking on a trailer as it is being hauled. This allows for the driver to capture the otherwise lost braking energy to charge a portable battery mounted on the trailer for use in charging other devices, or propelling the trailer using the electric motor instead of gasoline.

Spectrometer pH Measurement Testbench

Learn more

This pH measurement testbench serves as a proof of concept for a single measurement system within an upcoming compact device meant to analyze seawater across a broad range of properties while out at sea.
With the immense bulk of carrying separate equipment for multiple tests for seawater sample analysis, this system aims to combine multiple modes of measurement into a single system that can be used while out at sea to reduce error associated with needing to bring seawater samples to land before broad range measurement can be conducted.
The pH measurement testbench consists of an automated stirring and measurement system designed to measure the pH of a seawater sample. This system also stabilizes the temperature of the LED module to greatly reduce variance in intensity due to thermal effects.

PCB-Wound Electromagnet Design Automation

Learn more

This project focuses on the use of MATLAB to automatically generate code to import into KiCAD for rectangular, planar helix electromagnets. The parameters for these electromagnets can be varied to create electromagnets of arbitrary size, current capacities, and layer count. Additionally, the resultant magnetic field of these electromagnets can be calculated and saved to an external file.

Charged Particle Simulator

Learn more

This project focuses on simulating the movement of an arbitrary charged particle through an arbitrary magnetic and electric field. This simulator uses the Boris method and can simulate a single particle at a time or an array of particles with the same properties coming from the same direction, but spread along the plane orthogonal to the axis of travel.
